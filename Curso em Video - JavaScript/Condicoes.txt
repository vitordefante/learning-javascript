Estruturas condicionais 🎯


Exemplo da sintaxe:
if (condicao) {
	acao1 = true }

else {
	acao2 = true }


Condicoes Aninhadas 🐓
if (condicao) {
	acao1 = true
}

else {
	if (condicao) {
		funcao()
	}
}


Condicoes Multiplas 🖐
As condicoes multiplas precisam de um break ao fim de cada bloco.

swich (expressao) {
	case valor1:
	 function()
	 break
}
